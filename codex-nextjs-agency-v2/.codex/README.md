# Codex bootstrap

Copy `.codex/` and `AGENTS.md` into the root of a Next.js repository.

The default operating sequence is:

classify -> research -> design -> implement -> verify -> review -> document

Use the smallest sufficient workflow for the task.

## Skills

This bootstrap uses the open Agent Skills layout:

`.codex/skills/<skill-name>/SKILL.md`

A small scaffold helper is provided at:

`.codex/tools/create-skill.sh`
